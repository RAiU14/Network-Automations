✅ Phase 2 (End of improvements – 17/08 evening)
Context: Enhancements for usability, robustness, and reporting — while still logic-preserving for parsers.

Cisco_IOS_XE
get_ip_address → Strict IPv4 validation (0–255). If not in filename → content fallback.

_placeholder_entry (NEW) → For skipped/failed/Non-IOS-XE: returns full row with "Unsupported IOS" everywhere + Remark = "Non_IOS_XE".

append_to_excel → Always calls:

                 -add_summary_sheet (sheet order + rollups).
                    → Ensures workbook consistency on every write.

process_and_style_excel
Preserves existing Remark (e.g., "Non-IOS_XE") and only fills when blank.
Coerces percentages.

Self-heals PM sheet name → “Preventive Maintanance”.

add_summary_sheet
